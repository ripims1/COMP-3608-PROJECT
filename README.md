# Kaggle Competition: Predicting Customer Churn

This project is an attempt at the customer churn prediction problem using a dataset from a Kaggle competition.

## Dataset 

The dataset used for this project is sourced from Kaggle and is **too large to be included in this repository**.

## Notes

Please download the 'train.csv' and 'test.csv' files from the
[Customer Churn Prediction competition](https://www.kaggle.com/competitions/playground-series-s6e3/data),
before trying to run the model.
